# CoinMetric

Android-приложение для семейного планирования бюджета с локальной БД и базовой синхронизацией через Google-аккаунт.

## Что реализовано
- Локальная база данных на Room (категории, участники семьи, транзакции, лимиты по категориям, постоянные платежи).
- Семейный доступ (список участников с email Google).
- Категории расходов и фундамент для месячных лимитов по категориям.
- Постоянные платежи.
- Добавление транзакций с мини-калькулятором выражений (`1200+350/2`).
- Календарный просмотр операций по дням.
- Аналитический дэшборд (доходы, расходы, баланс, расходы по категориям).
- WorkManager-воркер для периодической синхронизации (заглушка) при наличии Google-аккаунта на устройстве.

## Технологии
- Kotlin + Jetpack Compose
- Room
- WorkManager
- Google Sign-In (получение аккаунта)
